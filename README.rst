Bit DNA
=======

bit_dna是一个用来把ATCG转化成二进制存储，并且能够读取已经保存的二进制文件到ATCG的包

转换规则:

+--------+---------+
|  碱基  |  二进制  |
+========+=========+
|   A    |   0b00  |
+------------------+
|   T    |   0b01  +
+------------------+
|   C    |   0b10  +
+------------------+
|   G    |   0b11  +
+------------------+
